# Clip Maker Frontend

React + TypeScript + Vite frontend for the video caption generator. 

## Features

- Upload video files or provide YouTube URLs
- Trim & crop video before processing
- Custom video player controls
- Real-time job status with WebSocket
- Caption editing interface
- Download processed video with captions

## Setup

```bash
npm install 
npm run dev
```

## Build

```bash
npm run build
```
